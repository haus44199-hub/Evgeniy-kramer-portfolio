## 📱 About this project

This is my **personal business card website** — a modern, fully responsive landing page that represents me as an Android developer. It showcases my flagship AI‑client project, my technical philosophy, and the tools I master.

The site is built as a single, self-contained HTML/CSS/JS file with **built‑in bilingual support** (Russian/English) and a clean, dark-themed UI that reflects my approach to development: fast, thoughtful, and user‑first.

---

## ✨ Key features

- **Dual language** — seamless RU/EN switching via JavaScript (no external dependencies)
- **Fully responsive** — works perfectly on desktop, tablet, and mobile
- **Dark theme optimized** — reduced eye strain, modern look
- **Smooth scrolling & micro‑interactions** — hover animations, tactile feel
- **No external frameworks** — pure semantic HTML5 + CSS3 + vanilla JS
- **Print‑friendly layout** — clean typography and accessible contrast
